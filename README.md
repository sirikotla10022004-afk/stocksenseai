# StockSense AI - Demand Forecasting Platform

StockSense AI is a premium full-stack demand forecasting platform that helps businesses predict future sales trends using advanced Machine Learning models (Prophet and ARIMA).

## 🚀 Features
- **AI-Powered Predictions**: Real-time demand forecasting using Prophet and ARIMA.
- **Interactive Dashboard**: Modern, dark-mode UI with dynamic charts (Chart.js/Recharts).
- **Trend Analysis**: Visualize top trending products and seasonal outlooks.
- **Dual-Layer Fallback**: Synthesizes intelligence locally if the server is offline.

## 🛠 Tech Stack
- **Frontend**: Next.js (React), Tailwind CSS, Framer Motion.
- **Backend**: Python (Flask/FastAPI), Prophet, Statsmodels, Pandas.
- **Deployment**: Optimized for Netlify (Frontend) and Render/Heroku (Backend).

## 📦 Installation

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python main.py`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
